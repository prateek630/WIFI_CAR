# ESP32 WiFi Controlled Car

A WiFi-controlled robotic car based on ESP32.

## Variants

### With PWM
- Speed control supported
- Smooth acceleration
- Better motor control

Location:
Code/With_PWM/

### Without PWM
- Simple ON/OFF motor control
- Easier for beginners

Location:
Code/Without_PWM/

## Hardware Used

- ESP8266
- Arduino Nano
- L298N Motor Driver
- DC Gear Motors
- Battery Pack
